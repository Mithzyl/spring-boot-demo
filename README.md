# Spring Boot 入门样例




## 项目简介

本项目作为Spring Boot入门样例，从浅到深用样例学习Spring Boot，并集成Spring Boot常用功能，包括logback日志，Aop日志

> 本demo用spring boot在网页上输出一个简单的hello world
